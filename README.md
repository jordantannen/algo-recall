# AlgoRecall

A spaced repetition app for tracking LeetCode problem reviews. 

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind
- **Backend**: Firebase (Firestore + Authentication)
- **Icons**: Lucide React
